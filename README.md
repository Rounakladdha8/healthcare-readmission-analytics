# Healthcare Readmission Risk Analytics

## Problem
Hospital systems aim to reduce 30-day patient readmissions to improve care quality and reduce operational costs. Identifying high-risk patient segments enables targeted follow-up care and intervention programs.

## Objective
Analyze hospital encounter data to identify factors influencing 30-day readmission risk and develop an analytics workflow for identifying high-risk patient cohorts.

## Dataset
100,000+ hospital encounters from a diabetes readmission dataset containing demographic, clinical utilization, and treatment information.

## Tools
Python (Pandas, Scikit-learn)  
SQL (analytical cohort queries)  
Power BI (interactive dashboard visualization)

## Methodology
1. Data exploration and feature analysis using Python
2. Feature engineering for readmission risk indicators
3. Logistic regression model for readmission probability estimation
4. SQL queries for cohort segmentation and operational reporting
5. Power BI dashboard for stakeholder decision support

## Key Insights
• Patients with higher prior inpatient utilization show significantly elevated readmission risk  
• Medication burden correlates with increased readmission probability  
• Younger adults (20-30) and elderly populations show higher risk segments

## Deliverables
• Exploratory analysis and predictive model (Python)  
• Analytical SQL queries for operational reporting  
• Interactive Power BI dashboard for monitoring readmission risk
