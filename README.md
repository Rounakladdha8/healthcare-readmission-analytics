# healthcare-readmission-analytics
Healthcare Readmission Risk Analytics

This project analyzes hospital patient encounter data to identify factors influencing 30-day readmission risk.

Tools

Python (Pandas, Scikit-learn)
SQL (analytical queries)
Power BI (dashboard visualization)

Dataset

100K+ hospital encounters from a diabetes readmission dataset.

Key Findings

• Patients with higher prior inpatient utilization show significantly higher readmission risk
• Medication complexity correlates with readmission probability
• Younger adults and elderly populations show elevated risk segments

Deliverables

• Exploratory data analysis and ML model (Python)
• Analytical SQL queries for cohort segmentation
• Interactive Power BI dashboard
